# API 参考

本节列出所有公开命名空间下的类型。点击左侧导航按类型浏览。

## 命名空间

### [CurlUnity.Http](CurlUnity.Http.yml)

HTTP 客户端和请求/响应模型。开始用 `CurlHttpClient` + `HttpRequest`,扩展方法看 `HttpClientExtensions` / `HttpRequestExtensions`。

### [CurlUnity.Diagnostics](CurlUnity.Diagnostics.yml)

可选的性能/连接统计。构造 `CurlHttpClient(enableDiagnostics: true)` 开启。
